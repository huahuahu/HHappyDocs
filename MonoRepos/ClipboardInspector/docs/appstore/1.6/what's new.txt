支持查看完整 UTType 链

ℹ️ 什么是UTType链？
苹果Uniform Type Identifier的层级结构，例如一个PNG图片的类型链会包含public.image(父类)和public.data(祖父类)，展示完整链条有助于理解数据格式的兼容性基础。


• Display the full UTType inheritance chain for copied items
(e.g., public.png → public.image → public.data)

ℹ️ ​What’s a UTType Chain?
Apple’s Uniform Type Identifier system organizes data types hierarchically. A PNG file’s chain (public.png → public.image → public.data) reveals its compatibility with apps supporting any ancestor type.
