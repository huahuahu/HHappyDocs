为开发者提供的剪切板调试工具。
查看原生剪切板与网页剪切板的内容。
预览原生剪切板的内容。
查看剪切板与网页、UITextView、UITextField的交互效果。

## English

A pasteboard debug tool for developers.
Support check content of native pasteboard and web clipboard.
Preview content of native pasteboard.
Can check interaction between clipboard and web view, UITextView and UITextView.
